## Prepare local environment -- 配置操作，只用做一次
```sh
$ git config --global user.name "your-name"
$ git config --global user.email "xxx@gmail.com"
```

## 常用的Git命令
```sh
# 获取远程服务器上的项目到本地
git clone https://github.com/openluopworld/wqlhtml.git

# 下面四条是用于在本地修改代码或温度后，提交到远程服务器
git add --all

git status

git commit -m "infomation about this commit"

git push
```

## 常用文件相关操作命令
```sh
linger@DESKTOP-26UPUD6 MINGW64 /

$ ls          // 查看当前目录下文件或目录

$ cd A-study-projects/

$ cd ../      // 在当前目录下回退一级，到父目录

$ cd ../../   //回退两级

$ touch a.log // 新建文件a.log
```
